# How to Run the App (Simple Version)

## 1. Open the Terminal
In your editor (VS Code), go to the top menu:  
`Terminal` -> `New Terminal`

## 2. Default Setup (Do this once)
Copy and paste these lines one by one into the terminal:

1. **Go to the project folder:**
   ```bash
   cd lms-web
   ```

2. **Install tools (only needed the first time):**
   ```bash
   npm install
   ```
   *(Wait for it to finish downloading)*

## 3. Start the App (Do this every time)
Whenever you want to work on the app, run:

```bash
npm run dev
```

## 4. Open in Browser
Click this link to see your app:  
[http://localhost:3000](http://localhost:3000)
